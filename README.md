# Figures For Paper — ISRGDF15 Project

This folder contains all scripts and data needed to reproduce the main figures in the paper.

---

## First download necessary RNAseq files and put into proper folders

GTEx v8 RNAseq data is available through the GTEx Portal (https://gtexportal.org/home/downloads/adult-gtex/bulk_tissue_expression). The Lifespan Study fibroblast dataset can be downloaded at https://columbia-picard.shinyapps.io/shinyapp-Lifespan_Study/ (GSE179848). RNAseq datasets from Labbé et al. are deposited in the Gene Expression Omnibus (GEO: accession numbers GSE273599, GSE273600, and GSE273601). 

1. Fibroblast Lifespan data should be placed in `Fibroblast_Lifespan/` folder. The file to download is: GSE179848_processed_cell_lifespan_RNAseq_data.csv.gz
2. Labbé et al. datasets should be placed in `Data/NatComms_K_Labbe_et_al/` folder.
3. GTEx
  i. Place GTEx_Analysis_2017-06-05_v8_RNASeQCv1.1.9_gene_tpm.gct.gz in `Data/gtex/`
  ii. Individual tissue .gz's should be placed `Data/gtex/All_Tissues_Indiv_Folders/` followed by the folder name of the tissue  
  iii. Place GTEx Annotations of Sample Attributes and Subject Phenotypes into: `Data/gtex/Insert_GTEx_SampleAttributes_and_SubjectPhenotypes_Here`
      - Access to Protected Access Data must be applied for at https://www.ncbi.nlm.nih.gov/projects/gap/cgi-bin/study.cgi?study_id=phs000424.v10.p2 
  

## How to Run

1. Open **`ISRGDF15_Paper_Figures.Rproj`** in RStudio. This anchors all `here()` paths to this folder.
2. Open **`00_Master_Script.R`** and run it. It will:
   - Process raw data → saves processed files to `Data/Fibroblast_lifespan/Processed/`
   - Generate all figures → saves to `Results/Figures/<FigureXX>/` or within FigureXX's folder

## Notes

- **`Insert_GTEx_SampleAttributes_and_SubjectPhenotypes_Here/`**: This folder contains GTEx phenotype/annotation files that are restricted by the GTEx data use agreement. Do **not** commit this to any public repository.
- **Runtime**: GTEx-based figures (3B, 3C, 3D, 4A) are slow because they process large expression matrices. Allow  ~.5 to several hours for a full run.
- **here() paths**: All paths in scripts use the `here` package and are relative to the project root (`ISRGDF15_Paper_Figures.Rproj`). Always open the `.Rproj` file before running scripts.
- **Latest version ran on:** R studio: Version 2026.01.0+392 (2026.01.0+392), MacOS: 14.5 (23F79), R: R version 4.4.1 (2024-06-14)
